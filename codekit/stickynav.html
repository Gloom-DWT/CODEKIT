CODEKIT

Qu'est ce que c'est ?

L'objectif de CODEKIT est de générer une série de composants d'interface permettant d'embellir un site web (animation, effet, ...) retrouvable sur mon Github.

Pour cette page nous trouverons un exemple au-dessus concernant le sticky nav.

Liste des composants :

  • Sticky nav
  • Back to the top
  • Progress bar
  • Burger menu
  • Système de tabs
  • Slider
  • Drop down
  • Parallax
